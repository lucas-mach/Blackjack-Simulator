FastAPI backend for the Blackjack card counting simulator
---

### Requirements
Python 3.10 or higher
<br><br>


### Setup and Installation
Navigate to the backend folder in your terminal: <br>
`cd backend` <br><br>
Create and activate a virtual environment (recommended): <br>
Windows: <br>
`python -m venv venv` <br>
`venv\Scripts\activate` <br><br>
MacOS/Linux: <br>
`python3 -m venv venv` <br>
`source venv/bin/activate` <br><br>
Install the required packages: <br>
`pip install -r requirements.txt` <br>
Or if you prefer to install directly: <br>
`pip install fastapi uvicorn[standard]`
<br><br>


### Running the API locally
With the virtual environment activated, run: <br>
`uvicorn main:app --reload --port 8000` <br><br>
The --reload flag makes the server restart automatically when you change code <br><br>
The API will be available at: http://127.0.0.1:8000
<br><br>


### Available Endpoints
**GET /health--** <br>
Simple health check endpoint to confirm the backend is running. <br>
Example response (JSON): <br>
`{
"status": "ok",
"message": "Backend is running successfully!",
"version": "0.1.0"
}`
<br><br>

**POST /simulate_hand--** <br>
Simulates one complete hand of blackjack using basic strategy (player hits <17, dealer hits <17). <br><br>

Interactive documentation (automatically generated by FastAPI): <br>
http://127.0.0.1:8000/docs     (Swagger UI - try clicking the health endpoint here)
<br><br>

### Stopping the Server
Press Ctrl + C in the terminal window where uvicorn is running.
